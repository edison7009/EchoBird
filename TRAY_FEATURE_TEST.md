# 系统托盘功能测试指南

## 新增功能

1. **首次关闭确认对话框**
   - 用户首次点击关闭按钮时，会弹出对话框询问关闭行为
   - 两个选项：
     - "直接关闭"：点击关闭按钮时直接退出应用
     - "最小化到托盘"：点击关闭按钮时隐藏窗口到系统托盘
   - 用户选择后会保存设置，下次不再询问

2. **托盘图标交互**
   - 左键单击托盘图标：显示/隐藏窗口（切换）
   - 右键单击托盘图标：显示菜单（显示 EchoBird、退出）

3. **托盘菜单国际化**
   - 托盘菜单会跟随用户在设置中选择的语言
   - 支持中文和英文

4. **设置界面**
   - 在设置对话框中可以修改"关闭窗口时"的行为
   - 两个选项：直接关闭 / 最小化到托盘

## 测试步骤

### 测试 1: 首次关闭确认
1. 删除设置文件（如果存在）：`~/.echobird/settings.json`
2. 启动应用
3. 点击窗口右上角的关闭按钮（X）
4. 应该弹出对话框询问关闭行为
5. 选择"最小化到托盘"
6. 窗口应该隐藏，托盘图标应该出现

### 测试 2: 托盘图标交互
1. 左键单击托盘图标
2. 窗口应该重新显示
3. 再次左键单击托盘图标
4. 窗口应该隐藏
5. 右键单击托盘图标
6. 应该显示菜单（显示 EchoBird、退出）

### 测试 3: 托盘菜单国际化
1. 打开设置对话框
2. 切换语言（中文 ↔ 英文）
3. 右键单击托盘图标
4. 菜单文本应该跟随语言变化

### 测试 4: 设置持久化
1. 在设置中修改"关闭窗口时"的行为
2. 关闭应用（通过托盘菜单的"退出"）
3. 重新启动应用
4. 点击关闭按钮
5. 应该按照之前的设置执行（不再弹出对话框）

### 测试 5: 完全退出
1. 选择"最小化到托盘"模式
2. 点击关闭按钮，窗口隐藏到托盘
3. 右键单击托盘图标
4. 选择"退出"
5. 应用应该完全退出，托盘图标消失

## 技术细节

- 首次关闭对话框使用 `ConfirmDialog` 组件
- 设置保存在 `~/.echobird/settings.json`
- 托盘菜单在语言切换时动态重建
- 关闭行为在 `TitleBar.tsx` 的 `handleClose` 函数中处理
