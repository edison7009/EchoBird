三步开始使用

1

下载安装

前往 GitHub Releases 下载最新版本的安装包，双击运行完成安装。支持 Windows、macOS 和 Linux 系统。

2

配置模型

打开 EchoBird，在"模型管理"页面添加你的 API Key（支持 Claude、OpenAI 等），或者部署本地模型。所有配置都有详细的引导提示。

3

安装工具

在"工具管理"页面，选择你需要的 AI 编程工具（如 Claude Code、OpenClaw），点击"一键安装"按钮，几秒钟即可完成安装和配置。
